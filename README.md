## Description

 - As interações com os desafios e submissões desenvolvidas no Postman podem ser importadas por esse [Link](https://www.getpostman.com/collections/04f2804b4ddb2777baf2);


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Migration

```bash
# generate migration
$ npm run migration:generate

# run migration
$ npm run migration:run

```
